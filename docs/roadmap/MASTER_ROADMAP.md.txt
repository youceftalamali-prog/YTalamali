# Project Goal

Transform AuraPost AI into a production-grade Enterprise SaaS platform for AI-powered Ecommerce Automation.

The platform must become stable, secure, scalable and commercially deployable.

No feature will be added unless it follows the official architecture.

# Development Rules

Rule 1

Never modify production code without documentation.

Rule 2

Never implement a feature without architecture approval.

Rule 3

Every feature must have:

- Backend
- Frontend
- Database
- API
- Tests
- Documentation

Rule 4

No temporary fixes.

Rule 5

No duplicated code.

Rule 6

No mock implementation inside production.
